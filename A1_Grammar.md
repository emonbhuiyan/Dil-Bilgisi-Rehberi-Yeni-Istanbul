# Ünite 1 – Merhaba (Hello)

## 1. Türk Alfabesi (Turkish Alphabet)

Türkçede 29 harf vardır. (There are 29 letters in Turkish.)

| Büyük Harf | Küçük Harf | Türü  | English Note                      |
| ---------- | ---------- | ----- | --------------------------------- |
| A          | a          | Ünlü  | vowel                             |
| B          | b          | Ünsüz | consonant                         |
| C          | c          | Ünsüz | pronounced like English “j”       |
| Ç          | ç          | Ünsüz | pronounced like “ch”              |
| D          | d          | Ünsüz | consonant                         |
| E          | e          | Ünlü  | vowel                             |
| F          | f          | Ünsüz | consonant                         |
| G          | g          | Ünsüz | hard g                            |
| Ğ          | ğ          | Ünsüz | soft g – lengthens previous vowel |
| H          | h          | Ünsüz | consonant                         |
| I          | ı          | Ünlü  | dotless i                         |
| İ          | i          | Ünlü  | dotted i                          |
| J          | j          | Ünsüz | like French “j”                   |
| K          | k          | Ünsüz | consonant                         |
| L          | l          | Ünsüz | consonant                         |
| M          | m          | Ünsüz | consonant                         |
| N          | n          | Ünsüz | consonant                         |
| O          | o          | Ünlü  | vowel                             |
| Ö          | ö          | Ünlü  | like German ö                     |
| P          | p          | Ünsüz | consonant                         |
| R          | r          | Ünsüz | rolled slightly                   |
| S          | s          | Ünsüz | consonant                         |
| Ş          | ş          | Ünsüz | like “sh”                         |
| T          | t          | Ünsüz | consonant                         |
| U          | u          | Ünlü  | vowel                             |
| Ü          | ü          | Ünlü  | like German ü                     |
| V          | v          | Ünsüz | consonant                         |
| Y          | y          | Ünsüz | consonant                         |
| Z          | z          | Ünsüz | consonant                         |

### Ünlüler (Vowels)

| Kalın Ünlüler | İnce Ünlüler |
| ------------- | ------------ |
| a, ı, o, u    | e, i, ö, ü   |

### Not / Exception

* Turkish does not use the letters Q, W, X.
* Capital I becomes ı, but capital İ becomes i.

| Büyük Harf | Küçük Harf |
| ---------- | ---------- |
| I          | ı          |
| İ          | i          |

This is different from English.

## 2. Bu / Şu / O – Kim? / Ne?

(This / That – Who? / What?)

| Word | Meaning | Usage            |
| ---- | ------- | ---------------- |
| bu   | this    | near the speaker |
| şu   | that    | a little farther |
| o    | that    | far away         |

### Kim? (Who?)

Used only for people.

| Soru    | Cevap        |
| ------- | ------------ |
| Bu kim? | Bu öğretmen. |
| Şu kim? | Şu öğrenci.  |
| O kim?  | O doktor.    |

### Ne? (What?)

Used for objects or things.

| Soru   | Cevap     |
| ------ | --------- |
| Bu ne? | Bu kalem. |
| Şu ne? | Şu masa.  |
| O ne?  | O çanta.  |

### Exception / İstisna

Wrong:

* Bu kim? → Bu masa.

Correct:

* Bu ne? → Bu masa.

Because “kim” is only for people.

## 3. Çoğul Eki: -lar / -ler (Plural Suffix)

| Last Vowel | Suffix | Example    |
| ---------- | ------ | ---------- |
| a, ı, o, u | -lar   | kitaplar   |
| e, i, ö, ü | -ler   | öğrenciler |

| Singular | Plural     |
| -------- | ---------- |
| kitap    | kitaplar   |
| öğrenci  | öğrenciler |
| çocuk    | çocuklar   |
| gözlük   | gözlükler  |

### With bu / şu / o

| Singular   | Plural            |
| ---------- | ----------------- |
| bu kitap   | bunlar kitaplar   |
| şu öğrenci | şunlar öğrenciler |
| o doktor   | onlar doktorlar   |

### Exception / İstisna

After numbers, Turkish does NOT use the plural suffix.

| Correct     | Wrong          |
| ----------- | -------------- |
| üç kitap    | üç kitaplar    |
| iki öğrenci | iki öğrenciler |
| beş ev      | beş evler      |

## 4. Soru Eki: mı / mi / mu / mü (Question Particle)

| Last Vowel | Question Form |
| ---------- | ------------- |
| a, ı       | mı            |
| e, i       | mi            |
| o, u       | mu            |
| ö, ü       | mü            |

| Statement  | Question      |
| ---------- | ------------- |
| Bu masa.   | Bu masa mı?   |
| Bu ev.     | Bu ev mi?     |
| Bu okul.   | Bu okul mu?   |
| Bu gözlük. | Bu gözlük mü? |

### Important Note

The question particle is always written separately.

| Correct     | Wrong      |
| ----------- | ---------- |
| Bu masa mı? | Bu masamı? |
| Öğrenci mi? | Öğrencimi? |

## 5. Değil (Not)

Used to make negative nominal sentences.

| Person | Form       |
| ------ | ---------- |
| ben    | değilim    |
| sen    | değilsin   |
| o      | değil      |
| biz    | değiliz    |
| siz    | değilsiniz |
| onlar  | değiller   |

| Positive      | Negative            |
| ------------- | ------------------- |
| Ben doktorum. | Ben doktor değilim. |
| O öğrenci.    | O öğrenci değil.    |
| Biz Türküz.   | Biz Türk değiliz.   |

---

## 6. Burası Neresi? (Where is this place?)

| Question       | Answer         |
| -------------- | -------------- |
| Burası neresi? | Burası okul.   |
| Şurası neresi? | Şurası banka.  |
| Orası neresi?  | Orası hastane. |

### Note

In daily speech, people sometimes shorten these forms:

* Burası → bura
* Şurası → şura
* Orası → ora

But in writing and in the book, use the full forms.

# Ünite 2 – Nerede? (Where?)

## 1. Bulunma Durumu: -DA / -DE (Locative Case)

Used to answer “Where?”

| Word Ending | Suffix    | Example        |
| ----------- | --------- | -------------- |
| a, ı, o, u  | -da / -ta | okulda, parkta |
| e, i, ö, ü  | -de / -te | evde, markette |

### Sert Ünsüz Rule (Consonant Hardening)

If the word ends in `ç, f, h, k, p, s, ş, t` use t.

| Word  | Correct |
| ----- | ------- |
| park  | parkta  |
| sınıf | sınıfta |
| ağaç  | ağaçta  |

| Wrong   | Correct |
| ------- | ------- |
| parkda  | parkta  |
| sınıfda | sınıfta |

---

## 2. Var / Yok (There is / There isn't)

| Sentence             | Meaning                          |
| -------------------- | -------------------------------- |
| Evde televizyon var. | There is a TV at home.           |
| Çantada para yok.    | There is no money in the bag.    |
| Sınıfta öğrenci var. | There are students in the class. |

### Question Form

* Evde bilgisayar var mı?
* Masada kitap yok mu?

# Ünite 3 – Ne Yapıyorsun? (What Are You Doing?)

## 1. Şimdiki Zaman: -(I)yor (Present Continuous)

| Last Vowel | Suffix |
| ---------- | ------ |
| a, ı       | -ıyor  |
| e, i       | -iyor  |
| o, u       | -uyor  |
| ö, ü       | -üyor  |

| Verb     | Present Continuous |
| -------- | ------------------ |
| okumak   | okuyor             |
| içmek    | içiyor             |
| konuşmak | konuşuyor          |
| gülmek   | gülüyor            |

### Full Conjugation – okumak

| Person | Form        |
| ------ | ----------- |
| ben    | okuyorum    |
| sen    | okuyorsun   |
| o      | okuyor      |
| biz    | okuyoruz    |
| siz    | okuyorsunuz |
| onlar  | okuyorlar   |

### Exception / İstisna

Some verbs change their stem.

| Verb   | Wrong    | Correct |
| ------ | -------- | ------- |
| gitmek | gitiyor  | gidiyor |
| yemek  | yemiyor* | yiyor   |
| demek  | demiyor* | diyor   |
| etmek  | etiyor   | ediyor  |

*These are irregular stem changes.

### Examples

* Ben kitap okuyorum.
* Sen kahve içiyorsun.
* O okula gidiyor.
* Biz Türkçe öğreniyoruz.

---

## 2. Negative Form of Present Continuous

Structure:
verb + m + (I)yor

| Positive  | Negative   |
| --------- | ---------- |
| geliyorum | gelmiyorum |
| okuyor    | okumuyor   |
| gidiyoruz | gitmiyoruz |

### Exception

The irregular stem still remains:

* gitmek → gitmiyor
* yemek → yemiyor

---

## 3. Present Continuous Questions

| Statement    | Question         |
| ------------ | ---------------- |
| Geliyorsun.  | Geliyor musun?   |
| Okuyor.      | Okuyor mu?       |
| Çalışıyoruz. | Çalışıyor muyuz? |

### Note

The question particle is written separately.

| Correct        | Wrong         |
| -------------- | ------------- |
| Geliyor musun? | Geliyormusun? |

---

## 4. Yönelme Durumu: -A / -E (Dative Case)

Used for “to / toward”.

| Word       | Result       |
| ---------- | ------------ |
| okul       | okula        |
| ev         | eve          |
| park       | parka        |
| üniversite | üniversiteye |

### Exception / Buffer Letter

If the word ends with a vowel, add y.

| Word   | Correct   |
| ------ | --------- |
| Ankara | Ankara'ya |
| araba  | arabaya   |
| okul   | okula     |

# Ünite 4 – Benim Dünyam (My World)

## 1. İyelik Ekleri (Possessive Suffixes)

| Person | Suffix        | Example   |
| ------ | ------------- | --------- |
| ben    | -(I)m         | kitabım   |
| sen    | -(I)n         | kitabın   |
| o      | -(s)I         | kitabı    |
| biz    | -(I)miz       | kitabımız |
| siz    | -(I)niz       | kitabınız |
| onlar  | -ları / -leri | kitapları |

### With Vowel Ending Words

If the noun ends with a vowel, add s before the suffix for “o”.

| Word  | Correct |
| ----- | ------- |
| araba | arabası |
| anne  | annesi  |
| oda   | odası   |

### Exception

Wrong:

* arabaı

Correct:

* arabası

---

## 2. Belirtme Durumu: -(y)I (Accusative Case)

Used for specific/direct objects.

| Word  | Accusative |
| ----- | ---------- |
| kitap | kitabı     |
| kalem | kalemi     |
| araba | arabayı    |
| anne  | anneyi     |

### Buffer Letter Rule

If the noun ends with a vowel, add y.

| Word  | Correct |
| ----- | ------- |
| araba | arabayı |
| masa  | masayı  |

### Exception

The accusative is not used with indefinite objects.

| Correct           | Wrong                              |
| ----------------- | ---------------------------------- |
| Kitap okuyorum.   | Kitabı okuyorum. (if not specific) |
| Bir elma yiyorum. | Bir elmayı yiyorum.                |

But if the object is specific:

* Kitabı okuyorum.
* Elmayı yiyorum.

# Ünite 5 – Zaman Zaman (Sometimes)

## 1. Belirli Geçmiş Zaman: -DI (Simple Past)

| Last Sound            | Suffix             |
| --------------------- | ------------------ |
| normal                | -dı, -di, -du, -dü |
| after ç,f,h,k,p,s,ş,t | -tı, -ti, -tu, -tü |

| Verb   | Past  |
| ------ | ----- |
| gelmek | geldi |
| okumak | okudu |
| açmak  | açtı  |

### Full Conjugation – gelmek

| Person | Form     |
| ------ | -------- |
| ben    | geldim   |
| sen    | geldin   |
| o      | geldi    |
| biz    | geldik   |
| siz    | geldiniz |
| onlar  | geldiler |

### Exception / İstisna

Some verbs change.

| Verb   | Correct Past |
| ------ | ------------ |
| gitmek | gittim       |
| etmek  | ettim        |
| tatmak | tattım       |

The last consonant doubles.

### Negative

* Gelmedim.
* Gitmedi.

### Question

* Geldin mi?
* Geldi mi?

# Ünite 6 – Çevremiz ve Biz (Our Environment and Us)

## 1. İsim Tamlamaları (Noun Compounds)

### Belirtisiz İsim Tamlaması

noun + noun + possessive suffix

| Structure           | Example       |
| ------------------- | ------------- |
| isim + isim + -(s)I | okul kapısı   |
| isim + isim + -(s)I | mutfak masası |

### Belirtili İsim Tamlaması

noun + -(n)in + noun + possessive suffix

| Structure         | Example         |
| ----------------- | --------------- |
| Ali'nin + arabası | Ali'nin arabası |
| evin + kapısı     | evin kapısı     |

### Exception

The second noun must always take a possessive suffix.

| Wrong       | Correct       |
| ----------- | ------------- |
| okul kapı   | okul kapısı   |
| mutfak masa | mutfak masası |

---

## 2. Daha / En (Comparatives and Superlatives)

| Structure        | Example                |
| ---------------- | ---------------------- |
| daha + adjective | Ahmet daha uzun.       |
| en + adjective   | Ahmet en uzun öğrenci. |

### Examples

* Bu ev daha büyük.
* İstanbul Türkiye'nin en büyük şehri.
